# BIT4138 Advanced Cryptography Practical Logbook (Weeks 1–5)

## Project Overview

This repository contains the practical implementations and supporting files for the **BIT4138: Advanced Cryptography** course. The project demonstrates the application of classical and modern cryptographic techniques using Python and related tools.

## Objectives

* Implement classical encryption algorithms.
* Explore pseudorandom sequence generation and randomness testing.
* Demonstrate symmetric encryption using AES.
* Demonstrate asymmetric encryption using RSA.
* Practice version control using Git and GitHub.

## Technologies Used

* Python 3
* Visual Studio Code (VS Code)
* Git
* GitHub
* OpenSSL
* PyCryptodome
* Windows PowerShell

## Weekly Activities

### Week 1

* Development environment setup
* Python installation and verification
* OpenSSL verification
* Git repository initialization

### Week 2

* Caesar Cipher implementation
* Vigenère Cipher implementation
* Encryption and decryption testing

### Week 3

* LFSR implementation
* Pseudorandom sequence generation
* Randomness testing
* RC4-style stream cipher simulation

### Week 4

* AES key generation
* File encryption and decryption
* Performance evaluation

### Week 5

* RSA key pair generation
* Public key encryption
* Private key decryption
* Secure message transmission and validation

## Learning Outcomes

Through this project, practical experience was gained in:

* Classical cryptography
* Stream ciphers and randomness analysis
* Symmetric key cryptography (AES)
* Asymmetric key cryptography (RSA)
* Secure software development practices
* Version control using Git and GitHub
